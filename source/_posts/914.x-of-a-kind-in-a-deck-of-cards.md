## [卡牌分组](https://leetcode.cn/problems/x-of-a-kind-in-a-deck-of-cards/)

给定一副牌，每张牌上都写着一个整数。

你需要选定一个数字 X（X >= 2），使得可以将整副牌按下述规则分成 1 组或更多组：

- 每组都有 X 张牌。
- 组内所有的牌上都写着相同的整数。

仅当你可选的 X >= 2 时返回 true。

<!-- more -->

示例 1：

> 输入：deck = [1,2,3,4,4,3,2,1]
> 输出：true
> 解释：可行的分组是 [1,1],[2,2],[3,3],[4,4]

示例 2：

> 输入：deck = [1,1,1,2,2,2,3,3]
> 输出：false
> 解释：没有可行的分组。

示例 3：

> 输入：deck = [1]
> 输出：false
> 解释：没有可行的分组。

提示：

- 1 <= deck.length <= 10^4
- 0 <= deck[i] < 10^4

## 解析

统计每个数字出现的次数，判断所有次数的最大公约数是否 >= 2。

```javascript
var hasGroupsSizeX = function (deck) {
  const count = new Map();
  for (const card of deck) {
    count.set(card, (count.get(card) || 0) + 1);
  }

  const values = [...count.values()];

  function gcd(a, b) {
    return b === 0 ? a : gcd(b, a % b);
  }

  let g = values[0];
  for (let i = 1; i < values.length; i++) {
    g = gcd(g, values[i]);
    if (g < 2) return false;
  }

  return g >= 2;
};
```

时间复杂度 O(n log C)，空间复杂度 O(n)。
