## [格雷编码](https://leetcode.cn/problems/gray-code/)

n 位格雷码序列是一个由 2^n 个整数组成的序列，其中每个整数都在 [0, 2^n - 1] 范围内，第一个整数是 0，每对相邻整数的二进制表示恰好一位不同。

<!-- more -->

## 解析

公式法：`gray(i) = i ^ (i >> 1)`。

```javascript
var grayCode = function (n) {
  const result = [];
  for (let i = 0; i < (1 << n); i++) {
    result.push(i ^ (i >> 1));
  }
  return result;
};
```
