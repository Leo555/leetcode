## [x 的平方根](https://leetcode.cn/problems/sqrtx/)

给你一个非负整数 x，计算并返回 x 的 算术平方根。结果只保留整数部分，小数部分将被舍去。

<!-- more -->

示例 1：

> 输入：x = 4
> 输出：2

示例 2：

> 输入：x = 8
> 输出：2
> 解释：8 的算术平方根是 2.82842...，由于返回类型是整数，小数部分将被舍去。

## 解析

```javascript
var mySqrt = function (x) {
  let left = 0, right = x;
  while (left <= right) {
    const mid = (left + right) >> 1;
    if (mid * mid <= x) { left = mid + 1; }
    else { right = mid - 1; }
  }
  return right;
};
```
