## [独特的电子邮件地址](https://leetcode.cn/problems/unique-email-addresses/)

每个 有效电子邮件地址 都由一个 本地名 和一个 域名 组成，以 '@' 符号分隔。除小写字母之外，电子邮件地址还可以含有一个或多个 '.' 或 '+' 。

例如，在 alice@leetcode.com 中，alice 是本地名，而 leetcode.com 是域名。

如果在电子邮件地址的本地名部分中的某些字符之间添加句点('.')，则发往那里的邮件将会转发到本地名中没有点的同一地址。

如果在本地名中添加一个加号('+')，则会忽略第一个加号后面的所有内容。

给你一个字符串数组 emails，我们会向每个 emails[i] 发送一封电子邮件。返回实际收到邮件的不同地址数目。

<!-- more -->

示例 1:

> 输入：emails = ["test.email+alex@leetcode.com","test.e.mail+bob.cathy@leetcode.com","testemail+david@lee.tcode.com"]
> 输出：2

示例 2:

> 输入：emails = ["a@leetcode.com","b@leetcode.com","c@leetcode.com"]
> 输出：3

提示:

- 1 <= emails.length <= 100
- 1 <= emails[i].length <= 100
- emails[i] 由小写英文字母、'+'、'.' 和 '@' 组成
- 每个 emails[i] 都包含有且仅有一个 '@' 字符

## 解析

对每个邮箱地址，分离本地名和域名，处理本地名中的 '.' 和 '+'，然后统计不同地址的数量。

```javascript
var numUniqueEmails = function (emails) {
  const unique = new Set();

  for (const email of emails) {
    const [local, domain] = email.split("@");
    const normalized = local.split("+")[0].replace(/\./g, "");
    unique.add(`${normalized}@${domain}`);
  }

  return unique.size;
};
```

时间复杂度 O(n * m)，其中 n 是 emails 的长度，m 是邮箱的平均长度。空间复杂度 O(n * m)。
