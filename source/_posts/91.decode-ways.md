## [解码方法](https://leetcode.cn/problems/decode-ways/)

一条包含字母 A-Z 的消息通过 '1' -> 'A', '2' -> 'B', ..., '26' -> 'Z' 进行编码。给你一个只含数字的非空字符串 s，请计算并返回解码方法的总数。

<!-- more -->

## 解析

```javascript
var numDecodings = function (s) {
  if (s[0] === '0') return 0;
  const n = s.length;
  const dp = new Array(n + 1).fill(0);
  dp[0] = 1; dp[1] = 1;
  for (let i = 2; i <= n; i++) {
    if (s[i - 1] !== '0') dp[i] += dp[i - 1];
    const two = parseInt(s.substring(i - 2, i));
    if (two >= 10 && two <= 26) dp[i] += dp[i - 2];
  }
  return dp[n];
};
```
