## [位1的个数](https://leetcode.cn/problems/number-of-1-bits/)

编写一个函数，输入是一个无符号整数，返回其二进制表达式中数字位数为 '1' 的个数。

<!-- more -->

## 解析

```javascript
var hammingWeight = function (n) {
  let count = 0;
  while (n) { n &= n - 1; count++; }
  return count;
};
```

`n & (n-1)` 每次消除最低位的 1。
