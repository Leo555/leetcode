## [有效数字](https://leetcode.cn/problems/valid-number/)

给定一个字符串 s，返回 s 是否是一个有效数字。

<!-- more -->

有效数字可以是整数或小数，后面可跟指数部分（'e' 或 'E'）。

提示：

- 1 <= s.length <= 20
- s 仅含英文字母、数字、'+', '-', '.'

## 解析

用状态机或正则匹配。

```javascript
var isNumber = function (s) {
  s = s.trim();
  const re = /^[+-]?(\d+\.?\d*|\.\d+)([eE][+-]?\d+)?$/;
  return re.test(s);
};
```
