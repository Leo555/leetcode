## [斐波那契数](https://leetcode.cn/problems/fibonacci-number/)

斐波那契数通常用 F(n) 表示，形成的序列称为斐波那契数列。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。给定 n，请计算 F(n)。

<!-- more -->

## 解析

```javascript
var fib = function (n) {
  if (n <= 1) return n;
  let a = 0, b = 1;
  for (let i = 2; i <= n; i++) { [a, b] = [b, a + b]; }
  return b;
};
```
