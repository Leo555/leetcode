## [完全平方数](https://leetcode.cn/problems/perfect-squares/)

给你一个整数 n，返回和为 n 的完全平方数的最少数量。

<!-- more -->

## 解析

```javascript
var numSquares = function (n) {
  const dp = new Array(n + 1).fill(Infinity);
  dp[0] = 0;
  for (let i = 1; i <= n; i++) {
    for (let j = 1; j * j <= i; j++) {
      dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
    }
  }
  return dp[n];
};
```
