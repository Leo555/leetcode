## [判断子序列](https://leetcode.cn/problems/is-subsequence/)

给定字符串 s 和 t，判断 s 是否为 t 的子序列。

<!-- more -->

## 解析

```javascript
var isSubsequence = function (s, t) {
  let i = 0;
  for (let j = 0; j < t.length && i < s.length; j++) {
    if (s[i] === t[j]) i++;
  }
  return i === s.length;
};
```
