## [下降路径最小和](https://leetcode.cn/problems/minimum-falling-path-sum/)

给你一个 n x n 的 方形 整数数组 matrix，请你找出并返回通过 matrix 的下降路径 的 最小和。

下降路径 可以从第一行中的任何元素开始，并从每一行中选择一个元素。在下一行选择的元素和当前行所选元素最多相隔一列（即位于正下方或者沿对角线向左或者向右的第一个元素）。

<!-- more -->

示例 1:

> 输入：matrix = [[2,1,3],[6,5,4],[7,8,9]]
> 输出：13

示例 2:

> 输入：matrix = [[-19,57],[-40,-5]]
> 输出：-59

提示:

- n == matrix.length == matrix[i].length
- 1 <= n <= 100
- -100 <= matrix[i][j] <= 100

## 解析

使用动态规划，从倒数第二行开始向上遍历，每个位置的最小路径和等于当前位置的值加上下一行相邻三个位置的最小值。

```javascript
var minFallingPathSum = function (matrix) {
  const n = matrix.length;
  const dp = new Array(n).fill(null).map(() => new Array(n).fill(0));

  for (let j = 0; j < n; j++) {
    dp[n - 1][j] = matrix[n - 1][j];
  }

  for (let i = n - 2; i >= 0; i--) {
    for (let j = 0; j < n; j++) {
      dp[i][j] = matrix[i][j] + Math.min(
        dp[i + 1][j - 1] ?? Infinity,
        dp[i + 1][j],
        dp[i + 1][j + 1] ?? Infinity
      );
    }
  }

  return Math.min(...dp[0]);
};
```

时间复杂度 O(n²)，空间复杂度 O(n²)。
