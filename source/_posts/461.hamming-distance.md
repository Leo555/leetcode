## [汉明距离](https://leetcode.cn/problems/hamming-distance/)

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

<!-- more -->

## 解析

```javascript
var hammingDistance = function (x, y) {
  let xor = x ^ y, count = 0;
  while (xor) { xor &= xor - 1; count++; }
  return count;
};
```
