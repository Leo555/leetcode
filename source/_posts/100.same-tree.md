## [相同的树](https://leetcode.cn/problems/same-tree/)

给你两棵二叉树的根节点 p 和 q，编写一个函数来检验这两棵树是否相同。

<!-- more -->

## 解析

```javascript
var isSameTree = function (p, q) {
  if (!p && !q) return true;
  if (!p || !q) return false;
  return p.val === q.val && isSameTree(p.left, q.left) && isSameTree(p.right, q.right);
};
```
