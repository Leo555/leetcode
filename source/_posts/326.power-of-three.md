## [3 的幂](https://leetcode.cn/problems/power-of-three/)

给定一个整数，写一个函数来判断它是否是 3 的幂次方。

<!-- more -->

## 解析

```javascript
var isPowerOfThree = function (n) {
  return n > 0 && 1162261467 % n === 0; // 3^19 = 1162261467
};
```
