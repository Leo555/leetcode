## [数字范围按位与](https://leetcode.cn/problems/bitwise-and-of-numbers-range/)

给你两个整数 left 和 right，表示区间 [left, right]，返回此区间内所有数字按位与的结果。

<!-- more -->

## 解析

```javascript
var rangeBitwiseAnd = function (left, right) {
  let shift = 0;
  while (left < right) { left >>= 1; right >>= 1; shift++; }
  return left << shift;
};
```

不断右移找到公共前缀。
